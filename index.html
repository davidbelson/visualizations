Visualizations

Enabling users to tell and share their own data-driven stories increases the reach and value of that data to a broader audience. I helped to define and design the visualizations shown below, and in some cases, used them in my written work.

Unfortunately, some of these visualizations are no longer active, but links are provided to those that are still available.

Oracle Internet Intelligence Map

A tool designed to democratize Internet analysis by exposing internal capabilities and data to the public. Updated in near real-time.

Connection Speed Trends

A tool that enabled users to create and share custom choropleth maps and trending graphs based on Akamai State of the Internet connection speed data. Was updated quarterly inconjunction with each new report.

State of the Internet iOS App

A mobile app that enabled users to create and share trending graphs based on Akamai State of the Internet connection speed data, as well as read State of the Internet reports on their mobile devices. Was updated quarterly inconjunction with each new report.

IPv6 Adoption - Country & ASN

A tool that provides insight into IPv6 adoption levels over time at a country and network provider (ASN) level, based on traffic to Akamai's global platform. Updated daily.

Akamai Internet Observatory

A tool that provided insight into mobile and desktop OS and browser adoption trends, based on Akamai Real User Monitoring (RUM) data. Was updated daily.

Akamai in 60 Seconds

A visualization intended to help make the massive amount of data and number of transactions processed by the Akamai platform more accessible by a wider audience. Was updated every five minutes.
